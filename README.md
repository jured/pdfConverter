# pdfConverter
Convert and merge files to pdf, at the moment only pdf and jpg files are supported



##Requied:
``` bash
node v0.12.0+
npm
pdfinfo
``` 

##How to setup:
``` bash
git clone https://github.com/jured/pdfConverter.git
cd pdfConverter
npm install
```

##Run:
``` bash
npm start
```
visit 127.0.0.1:3001 in your browser
